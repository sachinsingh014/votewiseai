# VoteWise AI Backend — Credentials Folder

This folder holds your Google Cloud service account JSON key.

## ⚠️ IMPORTANT

- This folder is listed in `.gitignore` and will NEVER be committed to Git.
- Copy your service account JSON file into this folder.
- Rename it to `service-account.json` (or update `GOOGLE_APPLICATION_CREDENTIALS` in `.env`).

## Required file

```
credentials/
└── service-account.json   ← Your downloaded JSON key goes here
```
